# Other Installation Options

## Manually with the file

[Releases Download Link](https://github.com/Viterkim/RhinosCanFly/releases/latest)

Download the zip from releases and extract the whole thing.

Open Rhino and click the top toolbar: Tools -> Options... -> Plugins -> Install (at the bottom) and select `RhinosCanFly.rhp`
